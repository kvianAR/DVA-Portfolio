# Aditya Ranjan - Developer Portfolio 🚀

A modern, responsive personal portfolio website built to showcase my full-stack development, AI automation, and data analytics projects.

## 🛠 Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion
- **Tooling**: Vite, PostCSS

## 🌟 Featured Projects
- **Uber Tableau** - Interactive Tableau dashboard tracking trip demand patterns, revenue trends, and peak-hour surges across city zones.
- **WhatsApp AI Store Analytics Assistant** - AI-powered WhatsApp bot for real-time sales insights and natural language querying.
- **E-commerce Website with Admin Panel** - Full-stack store with product management and complete admin dashboard.
- **Salon Marketing Website** - High-converting marketing landing page with smooth animations and SEO-optimised structure.

## 🚀 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/kvianAR/DVA-Portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd DVA-Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
